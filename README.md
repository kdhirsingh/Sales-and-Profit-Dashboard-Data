# 📊 Sales and Profit Dashboard (Excel)

This project presents an interactive **Sales and Profit Dashboard** built using Microsoft Excel.  
The dashboard analyzes sales performance, profit trends, and customer purchasing behavior using data visualization techniques.

---

## 📌 Project Overview

The dashboard transforms raw sales data into meaningful insights that help understand business performance.  
It provides a clear view of sales trends, product performance, and profitability.

---

## 🛠 Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization
- Dashboard Design

---

## 📈 Key Dashboard Insights

- Total Sales Overview
- Total Profit Analysis
- Monthly Sales Trend
- Product Line Performance
- Top Customers by Sales
- Year-wise Sales Analysis

---

## 📂 Dataset Information

The dataset contains the following fields:

- Order Number
- Quantity Ordered
- Price Each
- Sales
- Order Date
- Status
- Month
- Year
- Product Line
- Product Code
- Customer Name

---
Skills Demonstrated: 
Excel, Data Analysis, Dashboard Creation, Business Insights

## 🎯 Project Objective

The objective of this project is to create a visual dashboard that helps analyze business performance and supports data-driven decision-making.
